# Binary Search Trees

## Introduction
A Binary Search Tree (BST) is a tree data structure where each node has at most two children.
For every node, all values in the left subtree are less, all in the right subtree are greater.

## Key Operations
- Search: O(log n) average, O(n) worst
- Insert: O(log n) average
- Delete: O(log n) average

## BST Insert Algorithm
To insert key k:
1. If tree is empty, create root node
2. If k < current.key, go left
3. If k > current.key, go right
4. Repeat until null pointer found, insert there

## Practice Questions
Q1: What is the time complexity of BST search in the worst case?
A) O(1) B) O(log n) C) O(n) D) O(n log n)
Correct: C) O(n) — when tree is a linked list (degenerate)
